The project features 5 small command line games.


# Setup

```bash
make install
```

## Start

```bash
brain-games
```

## Run "is even" game

```bash
brain-even
``` 

## Run "calculator" game

```bash
brain-calc
``` 

## Run "get common divisor" game

```bash
brain-gcd
``` 
	
## Run "find the missing progression element" game

```bash
brain-progression
```	

## Run "is number prime" game

```bash
brain-prime 
```
